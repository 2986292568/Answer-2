# Answer

欢迎star，
本系统目前版本是离线版本，模拟做题都是通过写死的，在后期的维护中将通过搭建后台服务器，显现动态做题，以及不同学科的做题，当然，也是开源的，喜欢的小伙伴也可以继续关注，有你想不到的以为哦。
